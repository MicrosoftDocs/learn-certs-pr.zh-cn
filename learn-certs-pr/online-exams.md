# 关于 Pearson VUE 的在线考试

## Pearson VUE 在线监考考试介绍

Pearson VUE 是一家提供考试服务的公司，它是微软的合作伙伴。您可以在舒适的家中或办公室使用 OnVUE 在线交付参加许多 Microsoft Certification 考试。在线监考通过网络摄像头和麦克风监控您的考试经历，从而保持考试结果的完整性。
<br/><br/>

> [!VIDEO https://www.microsoft.com/en-us/videoplayer/embed/RE4IFBp]

## 参与国家/地区及所支持的语言

如果您的国家/地区有在线考试，那么在线考试的语言与您所在国家/地区的考试中心相同。请注意，接待员支持、监考支持和监考软件提供的语言数量有限。您将在考试注册的过程中选择该语言；它可能与您参加考试的语言不同。大多数考生只能选择参加具有英语接待员支持、监考支持和监考软件的在线考试，而不管考试语言是什么。

## 在您注册在线考试之前

1. 请确保您的计算机满足或超过最低系统要求（详细说明如下）。
2. 在您将要测试的同一台计算机和同一位置上完成所需的 [系统测试](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)。
3. 查阅下面的系统需求、技术提示及测试环境需求。
4. 请确认您的 ID 符合 Pearson VUE [ID Policy](https://home.pearsonvue.com/Policies/1S/English) 并且确认您的 Microsoft Certification 配置文件上的名字与您即将在参加考试时出示的由政府颁发的 ID 上的名字完全匹配。 如要阅读更多关于如何检查您的认证配置文件上名字的信息，请访问：[管理您的认证配置文件](/learn/certifications/manage-certification-profile).
5. 请用您的手机号码更新您的 Microsoft Certification 配置文件，其中包括国家代码，以防 Pearson VUE 需要在考试期间联系您。如果您在考试期间失去联系，这是 Pearson VUE 可以联系您的唯一方式。有关如何更新您的配置文件说明，请访问：[管理您的认证配置文件](/learn/certifications/manage-certification-profile).
6. 请阅读 [Non-Disclosure Agreement (NDA)](/learn/certifications/microsoft-exam-non-disclosure-agreement). 在考试开始时，您必须阅读并同意 NDA 和 General Terms of Use for Microsoft Certification Exams。一旦考试开始，您只有5分钟的时间阅读它。请在您参加考试前阅读它，以确保您完全理解这些要求，并能在不耗尽时间的情况下同意这些条款。如果您在规定的时间内不同意 NDA，你将不能参加考试。
7. 如果您想为您的考试请求便利设施，请您在安排您的考试**之前**向 Pearson VUE 提交便利设施请求。有关如何提交申请的详细资料，请访问: [请求 Pearson VUE 的便利设施](/learn/certifications/request-accommodations)。

## 技术提示

### 软件和权限

- 登录到您正在使用的计算机的用户帐户应有本地管理权限。有关启用管理权限的帮助，请参阅 [Windows](https://support.microsoft.com/en-us/windows/how-do-i-log-on-as-an-administrator-63267a09-9926-991a-1c77-d203160c8563) 和 [Mac.](https://support.apple.com/en-lk/guide/mac-help/mtusr001/mac) 文档
- 如有可能，请在个人电脑而不是工作电脑上进行测试，这是因为工作电脑上通常安装了会阻止 OnVUE 软件启动的软件。
- 您可能需要配置您的互联网安全及杀毒软件，使得 OnVUE Secure Browser 可执行文件成为例外。
- 一些杀毒软件会导致性能问题。请考虑在考试时禁用防病毒扫描。
- 您可能需要确保安全软件不会阻止 JavaScript, Adobe Air 应用程序或 Adobe Flash Player 扩展程序的执行。

### 网络

- 在线监考考试可能会被代理服务器、网络数据包检测/过滤或严格的网络安全配置所中断。
- 在参加在线监考考试时，如有可能，请使用硬线的互联网连接。
- 不要绑定移动热点；这是明确禁止的。

## 测试环境要求

为保持在线监考考试的完整性，我们执行特定的政策、实践和程序。

### 办公室或家庭设置

您的办公室或家庭环境必须符合以下要求：

- 办公室、会议室或家里的空间必须有墙，有一扇关闭的门，而且不受干扰。
																			
- 在您测试期间，其他人不得留在或进入该房间。

### 与接待员和监考员进行沟通

- **电话。** 在考试签到的过程中，接待员会询问您的电话号码，以便在您进行考试签到时与您沟通。在您完成签到手续后，你会被要求将手机放在够不到但可以听到的地方。
- **聊天窗口。** 大多数与接待员和监考员的交流都是通过聊天窗口进行的。
- **麦克风。** 在签到的过程中，如有必要，在您进行考试时，您可以通过麦克风与迎宾员和监考员交流。
- **语言。** 所有与迎宾员和监考员的交流都是用有限数量的语言进行的，这些语言可能与您参加考试用到的语言有所不同。

### 身份验证

- 你必须出示一份当前政府签发的 ID 表格，如下列之一：
  - 驾驶执照。
  - 军官 ID。
  - 护照。
  - 身份证（国家/州/省身份证）。
  - 外国人登记卡（绿卡、永久居民、签证）。
  - 由政府颁发的当地语言 ID （非罗马字符）——只有从考生正在测试的国家颁发的才被接受。
- 手机最适合捕捉所需的头像和 ID，但如果您没有手机，也可以使用网络摄像头。
- 您所提供的由政府颁发的 ID 上的名字必须与您的 Microsoft Certification 和 Pearson VUE 配置文件上的名字**完全**匹配。 
- 如果您正在更新或续签您的身份证明，那么请注意，在线监考的考试不接受包括 ID 续期表在内的合法姓名更改表。
- 18岁以下的未成年人可以出示有效的学生 ID 作为身份证明。考生的监护人也必须出示有效的 ID，并且在签到的过程中给予口头同意。

### 考试区域扫描

- 在开始考试之前，请确保所有额外的显示器均不插电并远离您，所有其他电脑都被关闭且显示器处于黑暗状态。
- 工作区域应远离所有材料，并且以下物品不允许在伸手可及的范围内：书籍、记事本、便利贴、打字笔记/纸，或钢笔、记号笔、白色书写板或铅笔等书写工具。
- 手机最适合拍摄所需的考试区域照片，但如果您没有手机，也可以使用网络摄像头。如果您用手机，这些照片会被上传，接待员会查看您的头像及 ID。该接待员正在寻找安全隐患，如果有什么需要在考试前解决的问题，会通知您。

### 禁止个人物品

- 在考试期间，您不能使用手提电脑或其他电子设备、寻呼机、手表、钱包、手提袋、帽子（或其他头罩）、包、外套、书籍、笔记，或任何其他未经特别批准的材料。
- 如果出现支持问题，监考员可能会尝试通过您在考试签到期间提供的手机号码联系您。请确保您的手机号码包含国家代码。将手机放在附近是可以接受的，但不要放在伸手可及的范围内。
- 您可能会被要求将您所有的口袋（裤子口袋和上衣/毛衣口袋)都翻出来，这样接待员就可以看到它们是空的。
- 如果您穿的是长袖衣服，那么您可能会被要求卷起袖子，以表示您的手臂上没有文字。
- 如果您系着领带，您可能会被要求解开领带，向接待员表明领带下面没有隐藏任何东西。
- 如果您的头发很长，遮住了耳朵，您可能会被要求将头发往后拉，向接待员表明您的耳朵上没有戴任何东西（比如蓝牙耳机）。

### 禁止休息或进食

- 考试期间不允许以任何理由进行休息。您了解并同意，如果您休息，您的考试成绩将被作废，并且不允许稍后继续考试。
- 考试期间禁止进食、吸烟及嚼口香糖。如果您在考试期间需要喝点什么，可以在一个干净的容器里放一杯水。

### 禁止考试帮助

- 在您考试期间，任何人不得进入考场。如果发生这种情况，您的考试将被终止。
- 在考试过程中，您不得接受其他个人的帮助或指导，并且您也不能让其他个人在考试期间看到电脑屏幕。
- 考试期间，您不能使用书籍或其他资源、草稿纸、可擦写白板或类似的东西，或书写工具，如钢笔、记号笔或铅笔。
- 监考员不能回答与考试内容相关的问题。如果您对考试内容有疑问，请在完成考试后联系 Microsoft。

### Facial Comparison Policy

- 通过注册在线监考的考试，您了解并同意 Pearson VUE 可能会在考试过程中使用人脸比对技术来验证您的身份。该技术会将您的面部图像与您身份证上的照片以及测试过程中捕捉到的面部图像进行比较，以帮助我们进一步开发、升级并改进此应用程序。如果您不同意这样，请在一家考试中心安排考试时间。

### 考试期间的录音

- 在考试时，监考员会通过视频和音频持续对您进行监控。整个考试过程都将被记录下来。监控内容将包括您的面部（眼球运动、头部运动）、声音、您所处的物理房间以及考试期间的位置。如果您不希望被录下（通过视频或音频），请立即通知监考员。您将不能继续进行考试。

## 保密协议

- 与任何其他 Microsoft Certification 考试一样，您必须阅读并同意我们的 [NDA](/learn/certifications/microsoft-exam-non-disclosure-agreement) 中所规定的条款。一旦考试开始，您只有5分钟的时间阅读该协议，因此我们建议您在参加考试前阅读它，以确保您完全了解这些要求，并能在不耗尽时间的情况下同意这些条款。如果您在规定的时间内不同意 NDA，您将无法参加考试。
- 您还必须遵守考生的行为要求。这包括但不限于以下内容： 
  - 你不得以任何方式复制或记录考试中的问题或答案。
  - 您不得与他人分享或讨论在考试中所看到的问题或答案。
- 如果您不遵守 [Candidate Rules Agreement](https://www.pearsonvue.com/rp/rp_candidate_rules_agreement.pdf) 或者涉嫌作弊或篡改来自 Pearson VUE 或其分包商的软件，或者如果考试期间发现有除您以外的人在您所处的房间内，监考员将会立即终止考试，并向 Microsoft 报告该违规行为，并同时提供音频和/或视频证据。您的考试可能会被视为无效，并且 Microsoft 可能会采取其他措施，比如取消您的认证和/或禁止您使用 Microsoft Certification Program。

## 启动考试和签到流程

启动和签到流程大约需要15分钟。

- 在开始考试之前，请查阅 OnVUE [FAQs](https://home.pearsonvue.com/microsoft/onvue) 以获取故障排除技巧。
- 在考试当天，在考试期间您即将使用的设备和网络连接上再次运行 [系统测试](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)。互联网带宽可以随时改变。
- 以管理员身份登录到您的计算机，这样您就可以确定是否有任一后台进程正在运行并将其停止。
- 关闭浏览器弹出窗口阻止程序。
- 确保您的手机在即时测试空间之外，但在听力所及范围之内，以防监考员需要联系您。
  - 确保您的 Microsoft Certification 配置文件随着您的手机号码一起更新，手机号码应包括国家代码，以防 Pearson VUE 需要在考试期间联系您。

开始您的测试环节并与接待员联系：

1. 使用与您的 Microsoft Certification 配置文件相关的电子邮件登录到您的 [Certification Dashboard](https://aka.ms/certdashboard)。（如果首先在此步骤中无法访问 Microsoft Certification Dashboard，请使用 Incognito 或 InPrivate 浏览器再次尝试。如果您仍然无法访问您的 Certification Dashboard，请使用 [认证支持](https://aka.ms/mcpforum) 打开一个案例，一位论坛版主将提供帮助。）
2. 我们建议提前15分钟开始签到流程。在您预约的时间之后，您将只有15分钟的时间来开始考试流程。
3. 在右侧的“预约部分”，找到您的考试预约并选择“开始在线考试”。您将被重定向到 Pearson VUE。
4. 选择 “Purchased Online Exams” 下面的考试名称。
5. 选择 “Begin exam” 并继续进行自助签到流程，同时等待监考员与您联系。（注意：您必须选择 “Begin exam” 以启动考试；如果您不这样做，考试就不会启动。） 
6. 如果您在预定的考试时间30分钟后没有连接到监考员，则连接就会丢失。在原定考试预约后的24至30小时内，您可以重新安排您的考试。只需使用与您的 Microsoft Certification 配置文件相关的电子邮件登录到 [Certification Dashboard](https://aka.ms/certdashboard) 以重新安排。我们建议您尽可能自助，但是您也可以联系 [Pearson VUE 支持](https://home.pearsonvue.com/microsoft/contact) 寻求帮助以重新安排考试。受 COVID-19 影响，响应时间比正常情况要慢（最多5天），请耐心等待。
7. 自助签到完成后，接待员会进行身份验证和房间扫描。如果在这些工件的接待员审查期间没有发现任何问题或关注事项，您将被安排在监考员的队列中。监考员一有空，您就可以开始考试了。在接待员对所提供的工件有疑问的情况下，接待员将与您一起解决这些问题。
8. 在这个流程完成后，考试将会开始。您的监考员不会在考试期间联系你，除非有问题。
9. 如果在考试交付的过程中出现了技术问题，监考员会首先尝试通过聊天与您联系。如果这样不起作用，监考员会通过电话联系您。
10. 如果考试后有问题，[请联系 Pearson VUE 支持。](https://home.pearsonvue.com/contact) 请使用由您的监考员提供的“事件报告”编号。

## 删除监考员软件
