# 关于在线考试

您可以在家中或办公室舒适地在线参加任何基于角色的考试或基础知识考试，同时由监考人员通过网络摄像头和麦克风进行监考。（不在线提供通过K12学校购买的基础知识与微软技术专员 (MTA) 考试。）如果您有兴趣参加在线考试，并希望在注册之前更好地了解在线考试过程，则此页面将解释详细信息。您也可以在[YouTube视频](https://www.youtube.com/playlist?list=PLahhVEj9XNTceGnF19qSjYxYzS4S_Wlmf)上观看视频的英语版本或或配有阿拉伯语、法语、德语、葡萄牙语或西班牙语字幕的版本。

3. **查看以下的技术、系统、环境和考试礼仪要求。**

## 系统要求

在线考试可能会因代理服务器、网络数据包检查/过滤或严格的网络安全配置而中断。

登录到考试工作站的 Windows 用户帐户必须具有本地管理权限。

部分防病毒软件可能导致性能问题。请在参加考试时禁用防病毒扫描。

您可能需要保证安全软件不会阻止 JavaScript、Adobe Air 应用程序或 Adobe Flash Player 扩展的执行。

您可能需要配置 Internet 安全和防病毒软件，对 Pearson Secure Browser 可执行文件进行例外处理。

## 考试期间的礼仪

为保证在线/网络摄像头监考环境下的考试完整性，我们将强制实施一些具体政策、做法和规程。

### 办公室或家庭环境

您的办公室或家庭环境必须符合以下要求：

- 办公室或会议室必须有墙壁，将大门关好，避免受到打扰。
- 如果是在家里参加考试，房间必须不受打扰。
- 在考试期间，房间内不能有任何第三方。

与接待员和监考员沟通：  

- **电话。** 在考试签到期间，接待员将询问您的电话号码，以便在考试签到过程中与您沟通。完成签到流程后，您需要将手机放在伸手够不到的位置，但不要太远，要保证足以听到声音。
- **聊天窗口。** 与接待员和监考员的大多数沟通都是通过聊天窗口完成的。
- **麦克风。** 在考试签到期间以及在考试过程中（如有必要），您可通过麦克风与接待员和监考员沟通。
- **语言。** 与接待员和监考员的所有沟通都应通过英语完成。

身份验证：

您需要提供现行政府颁发的身份类证件，如下列中的一种：

- 驾驶证
- 军人证
- 护照
- 身份证（国家/州/省发放的身份证）
- 外国人居留证（绿卡、永久居留证明、签证）。
- 政府颁发的当地语言身份证（非罗马字符） — _仅接受考生参加考试时所在国家/地区颁发的身份证_

网络摄像头会拍摄一张您和您的身份证件照片。

身份证件上的姓名必须与您的 Microsoft 和 Pearson VUE 个人资料中的姓名相符。

不到 18 岁的未成年人可以出示有效学生证来证明身份。考生的监护人也必须出示有效身份证件，并在签到流程中给予口头同意。

考试区域扫描：

要求您使用网络摄像头或通过举起和旋转笔记本电脑，从左到右缓慢地扫描以显示周围的墙壁，并上下扫描以显示地板和天花板。如果您无法完成考试区域扫描，则不允许进行下一步操作或参加考试。

随后，您需要扫描整个考试区。监管人查看安全隐患，包括但不限于

- 额外的显示器（必须将其插头拔下，且不得朝向您的方向）
- 额外的电脑（必须予以关闭，且显示器必须是黑屏）
- 壁画/巨幅装饰画（必须检查这些物品）

考试区域应该清除所有材料，包括手臂不能触及的以下物品：书本，笔记本，便笺，打印的笔记/纸张，或书写工具（如圆珠笔，记号笔或铅笔）。白板上的内容应擦拭干净。

### 不得使用私人物品

- 在考试过程中，不得使用以下类型的私人物品：手提电脑或其他电子设备、寻呼机、手表、钱包、手提包、帽子（或其他遮住头部的衣物）、包、外套、书籍、笔记或其他任何未明确许可的材料。
- 您需要翻开所有衣袋（裤子口袋和上衣/卫衣口袋），让接待员看到其中没有物品。
- 如果您穿着长袖衣物，您需要卷起袖子，保证您的手臂上没有小抄。
- 如果您佩戴领带，则需要抬起领带，向接待员展示领带下没有藏匿任何物品。
- 如果您长发盖耳，则需要将头发向后拨，向接待员展示耳朵上没有佩戴任何设备（例如蓝牙耳机）。

### 不得中断/进食/饮水

- 考试期间不允许任何原因的中断。您理解并同意，如果您中途休息一下，您的考试分数将失效，您将不允许在稍后继续考试。
- 考试期间禁止进食、饮水、吸烟和嚼口香糖。

### 不得有他人协助考试

- **在考试期间，不得有第三方进入考试房间。如若出现此类情况，您的考试将立即终止。**
- 在考试期间，您不得接受他人的协助或辅导，不能让他人看到显示考试题的电脑屏幕。
- 在考试期间，不得使用与考试相关的材料，禁止使用便笺纸、可擦除白板或其他书写物品。在考试期间，您不得使用任何书写工具，例如钢笔、马克笔或铅笔。
- 监考员不能解答与考试内容有关的任何问题。如果对考试内容有疑问，请在完成考试后联系 Microsoft。

其他重要信息

- 不得使用任何手段复制或记录考试题或答案，不得与他人讨论在考试中看到的考试题或答案。
- 参加考试即表示您确认：在考试期间，监考员将通过视频和音频持续监视您的行动；并且您确认和同意，在考试期间通过音频和视频方式录下您的面容、声音、您所在的房间以及考试地点。您承认，在考试期间，您在考试地点无隐私权，并且放弃任何有关隐私权的全部声明。如果您不希望被录音录像（包括视频和音频），请立即告知监考员您不想继续考试。
- 如果您不遵守[考生规则协议](https://www.pearsonvue.com/rp/rp_candidate_rules_agreement.pdf)或者被怀疑作弊或篡改 Pearson VUE 或其分包商的软件，或者在您考试期间所在的房间内检测到第三方的存在，监考员将立即终止考试，并将违规情况连同证明违规的音频和/或视频证据上报给 Pearson VUE 和 Microsoft。您的考试将立即作废，Microsoft 可能采取其他措施，例如收回您的证书，禁止您参加未来的所有 Microsoft 认证计划。

## 启动考试和考生签到流程

启动和签到流程大约应占用 15 分钟的时间。

- 关闭浏览器弹出窗口阻止程序。
- 务必确保您拥有电脑的管理员权限，以便下载软件。

要开始考试并与接待员连线：

- 登录到[您的权益和考试控制面板](https://www.microsoft.com/learning/dashboard.aspx)。
- 在右侧的**预约**部分中，选择**开始预先安排好的在线监考考试**。
- 随即将会显示您预先安排好的考试。选择考试链接，然后选择**开始考试**。

**注意**您可能要等待 5 分钟才能与接待员连线。

与接待员成功连线后：

- 准备好驾驶证、护照或其他可接受的身份证明，以便验证您的身份。
- 接待员将解说有关考试行为守则、考试环境和考试期间的沟通流程。与接待员和监考人的所有沟通都将以英语进行。
- 您将被要求关闭所有非必要应用程序（例如电子邮件或聊天程序）。

这个过程完成后，考试将开始。除非出现问题，否则监考员在考试期间不会与您联系。

如果在考试过程中出现技术问题，监考员将首先尝试通过聊天与您联系。如果这不起作用，监考员将通过电话与您联系。

如果考试后出现问题，请联系Pearson VUE。请留好监考员提供给您的意外事故报告号码。

### 获取分数报告，然后删除监考软件

可以从[您的权益和考试控制面板](https://www.microsoft.com/learning/dashboard.aspx)获取分数报告。

- 登录[www.microsoft.com/learning](https://www.microsoft.com/learning)。
- 要查看您的考试历史，请转到[您的权益和考试控制面板](https://www.microsoft.com/learning/dashboard.aspx)并选择**查看我的分数报告**。在这里，您可以保存并打印分数报告。
- 您的考试结果（通过或未通过）将转发给 Microsoft 进行处理。
- 当考试结束时，应删除所有可执行文件和辅助文件，包括**Pearson VUE安全浏览器**图标。要从桌面上删除此图标，请右键单击图标，选择**删除**，然后选择**是**。
- **[删除Pearson VUE安全浏览器的可执行文件](https://windows.microsoft.com/en-us/windows7/uninstall-or-change-a-program)**。

## 注册在线考试

在参与国家/地区，提供基于角色的考试或基础知识考试。如果您所在的国家/地区提供在线考试，则有以在您的国家/地区考试中心所支持的所有语言提供的在线考试。请注意，接待员支持、监考员支持以及监考软件目前仅使用英语。（不在线提供通过K12学校购买的Microsoft Office专员(MOS)考试、基础知识考试与微软技术专员(MTA)考试。）如要注册，请从[考试列表](https://www.microsoft.com/learning/exam-list.aspx)中选择您希望参加的Microsoft在线考试，然后选择**安排考试**按钮。如果在您的国家提供有在线考试，您可以在考试注册过程中选择该选项。

## 常见问题解答

查找有关在线监考考试的常见问题解答。

### 如何了解关于在线考试的更多信息？

微软发布了一个简短的视频，展示了如何容易地参加在线考试。可以在[YouTube 播放列表](https://www.youtube.com/playlist?list=PLahhVEj9XNTceGnF19qSjYxYzS4S_Wlmf)上观看此视频的英语版本或配有阿拉伯语、法语、德语、葡萄牙语或西班牙语字幕的版本。

### 我要怎样判断在线监考考试是否适合我？

首先完成要求的[系统检测](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)，确认您的电脑满足我们的要求。阅读了解有关[考试期间的礼仪](https://www.microsoft.com/learning/online-proctored-exams.aspx#protocol-label)的相关信息。如果您有任何疑虑或保留意见，建议您安排在考试中心参加考试。

### 如果我的系统未通过系统检测，我是否仍能参加在线考试？

不可以。如果您的系统未通过系统测试（包括您拍摄的身份证件照片是否清晰），您不应注册在线考试。

### 如何注册在线考试？

如要注册，请从[考试列表](https://www.microsoft.com/learning/exam-list.aspx)中选择您希望参加的考试，然后选择**安排考试**按钮。如果在您的国家提供有在线考试，您可以在考试注册过程中选择该选项。

### 如果不在考试中心考试，那岂不是更容易作弊？在线考试是否安全？

无论考试方式如何，我们会竭力保证考试的完整性。当您参加在线考试时，您必须首先安装一个安全的浏览器，在整个测试过程中执行严格的保护。此外，Pearson VUE监考员通过网络摄像头和麦克风在整个考试过程中对您进行监考。如果怀疑存在作弊行为，我们还会检查考试过程的视频。我们非常擅长检测作弊行为，所以我们不建议这样做 - 我们会收回作弊者的证件，取消他们的认证，并禁止他们以后参加考试。（[阅读我们的反盗版项目经理Kerri Davis的观点](https://www.microsoft.com/en-us/learning/community.aspx)。）

### 如果我要参加多个认证考试，是否需要选择一种单一的考试交付方式？

不需要，您可以混合搭配考试方式（在线或面对对考试中心），因为考试是相同的。

### 在考试开始并为我分配监考员之后，我需要在多长时间内完成考试？

您的考试时间与在 Pearson VUE 考试中心参加考试时相同。

### 在线考试比在考试中心的考试难吗？

否。考试内容都是相同的。

### 可以在参加在线考试时做笔记吗？

是。白板功能可输入笔记或绘制图形。您可以在考试期间通过单击屏幕顶部的橡皮擦图标访问白板。

### 在线考试是否支持多种语言？

是。如果您所在的国家/地区提供在线考试，则有以在您的国家/地区考试中心所支持的所有语言提供的在线考试。请注意，监考软件，接待员支持和监考人支持只能用英文进行。

### 如果我在在线考试开始之前、期间或之后遇到技术问题，我该联系谁？

有关注册问题，请使用注册期间提供的 Pearson VUE 客户服务链接。如果在考试进行期间遇到技术问题，考试监考员将帮助您尝试解决问题。如有必要，监考员会提供一个意外事故报告号码作为参考，以便您在后续联系[Pearson VUE客户服务](https://www.pearsonvue.com/microsoft/contact)。

### 签到过程中需要提供哪些类型的身份证明？

请查看当前的[身份证明政策](https://home.pearsonvue.com/Policies/1S/English)。如果您正在更新或续签您的身份证明，请注意，在线考试不接受合法姓名更改表格（包括身份证更新表格）。

### 我应如何更改合法姓名？

您的姓名必须与您在签到过程中提供的身份证明准确匹配。保存考试注册个人资料后，您只能通过联系您的[Microsoft地区服务中心](https://www.microsoft.com/learning/help)更改合法姓名。

### 我预约了一个在线考试。我该如何开始？

到达预约时间之后，您必须在 15 分钟内进入考试。15分钟过后，将不允许进入考试过程。

如要开始考试，请登录您的[权益和考试控制面板](https://www.microsoft.com/learning/dashboard.aspx)并点击**开始一个预先安排的在线监考考试**。您将转到一个 Pearson VUE 网页，从中将看到您预约的考试，随后即可开始考试。

### 我能在何处找到考试政策列表？

请参看适用于所有Microsoft认证考试的[考试政策和常见问题](https://www.microsoft.com/learning/certification-exam-policies.aspx)。
